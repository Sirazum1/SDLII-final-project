All about DropNote

“DropNote” is the project used for sharing notes using the website. Notes that the user wants for a particular subject can be found in the website. This project is written in Python Django and SQLite. Firstly, the user needs to sign up and then, they can log in to the system.

DropNote is a open source you can Download and edit as per you need.
